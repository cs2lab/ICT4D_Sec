Windows OS

The Windows Operating System contains a number of security features and configuration settings that can improve and harden the security posture of the system by applying security updates, filter network communications, examine for malicious files and applications, create backups of important configuration settings, controlling remote access enable improved and many more.

Instructions
Tasks

Perform the following tasks.

  1. Navigate to the Control Panel (accessible through the Start button) and click on the System and Security option. Then, click on the Windows Update option. The Operating System feature that is taking care of updating the system with security as well as other software updates has been turned off. Press the 'Turn on automatic updates' button in order to activate this feature. The system will start prompting you with notifications of software updates that your system is lacking. In the context of this lab assignment, you are requested to wait until you install all of the updates but instead you can proceed on the following steps.
  2. Return to the 'System and Security' page of the Control Panel and click on the Windows Firewall option. The firewall has also been deaativated. You can press the 'Use recommended settings' button in order to activate the firewall with a set of predefined rules

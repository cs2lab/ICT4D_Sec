Part 3

Security Hardening
Intro duction

Welcome to the Third part of the assignment on how to harden a system with respect to its security posture. Systems are often found with default or sloppy configuration settings that reduce the system's security. Various steps can be taken to enhance the system's security, a process often called 'hardening'.

The objectives of this part
Obje ctives

  • - Learn how to reduce the attack surface of a system
  • - Learn how to apply patches in a system
  • - Learn about common security-related settings and features of an Operating System like Windows such as firewalls and antivirus

Tools to be used
Tools

Windows OS

The Windows Operating System contains a number of security features and configuration settings that can improve and harden the security posture of the system by applying security updates, filter network communications, examine for malicious files and applications, create backups of important configuration settings, controlling remote access enable improved and many more.

Instructions
Tasks

Perform the following tasks.

  1. Let us start by employing an automated method for assessing the current system state in terms of security. Open the 'Microsoft Baseline Security Analyzer' tool that can be found on your Desktop. Click on the 'Scan a computer' option and review the scan options. You can use the default options and proceed by clicking on the 'Start Scan' button. Review the results regarding the current security status of the system.
  2. Navigate to the Control Panel (accessible through the Start button) and click on the System and Security option. Then, click on the Windows Update option. The Operating System feature that is taking care of updating the system with security as well as other software updates has been turned off. Press the 'Turn on automatic updates' button in order to activate this feature. The system will start prompting you with notifications of software updates that your system is lacking. In the context of this lab assignment, you are NOT requested to wait until you install all of the updates but instead you can proceed on the following steps.

    • Why is it important to keep the system up-to-date with security patches?
    • Can you describe a case where a system update may create more security issues than solving?
    • Is the update feature provided by the Operating System enough to protect the system or additional actions may be needed and why?

  3. Return to the 'System and Security' page of the Control Panel and click on the Windows Firewall option.The firewall is responsible of inspecting and ensuring that only allowed network connections are established by the system both in terms of outbound network conversations (e.g. the system accessing a Web site) as well as inbound (e.g. a remote user accesses a system's service). The rules that the firewall has to enforce are mostly defined in terms of networking concepts such as IP address and TCP/UDP ports but with respect to certain applications as well in the case of host-based firewall software. The firewall has also been deactivated. You can press the 'Use recommended settings' button in order to activate the firewall with a set of predefined rules. You can click on the Advanced Settings menu option on the left and examine some of the default inbound and outbound rules. As an example, let us add a new rule that disallows the Internet Explorer application from connecting to the Internet. Click on the Outbound Rules and select the 'New Rule...' option. Select the Program menu option and after Next use the Browse button to locate the iexplore executable that is located under the C:\Programm Files\Internet Explorer path. After Next select the 'Block the connection' option, click Next and Next and finally specify a name for the rule and click on Finish. Attempt to visit a website with both the Internet Explorer and the Firefox browsers and notice the difference. In the end, you can delete the Rule through the Delete option.

    • Consider a system that is connected to the Internet but the organization wants to use it as a Web server only for its employees. How can a firewall be employed to support such a case?
    • In most cases, firewalls are configured to block unwanted inbound connections to system ports and services but little attention is given to setting up a policy for outbound connections? How can outbound connections enhance the security of the system?

  4. Protecting the system from files and applications of malicious intent is a challenging task. There are a number of specialized programs that are able to scan the system in order to identify and eradicate instances of malware and can even protect you on real time as you download and open files downloaded from the Web or from email attachments. Navigate again to the Control Panel and select the View by Small Icons option on the top right of the window. Click on the Windows Defender option.Click on the Scan tooltip in the top menu in order to perform a quick scan.
  5. After the scanning process finishes, a malicious file, purposefuly copied to your system, must have been identified. Click on the 'Review detected items' in order to display more information about the detected file. You can proceed by cleaning the system from the malicious file
  6. Click on the Tools menu option on the top and then select Options. Enable the 'Automatically scan my computer' in order to configure the tool to perform an antimalware check of the system periodically. Switch to the Real-time protection menu option on the left in order to configure the tool to scan files downloaded over the Web. Optionally, in the Advanced menu option, you can also enable scanning e-mail attachments as well as removable drives such as USB sticks. Press the Save button to store your settings.

  7. Open Internet Explorer and visit the following URL: http://www.eicar.org/85-0-Download.html and download the eicar.com file. This file is a designated test file for checking antimalware programs. You should receive a notification by your system that a malicious file has been detected. Proceed by cleaning the system.

    • Why is it important to have an antivirus installed on a system?
    • Why is it important to keep the antivirus software updated?
    • What can be some limitations or concerns when using antivirus software?
